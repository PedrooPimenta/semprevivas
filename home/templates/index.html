O que é Sempre Vivas?

Sempre Vivas é um projeto de desenvolvimento de software, que visa a construção de um sistema de gerenciamento de espécies de erva-flores, com o objetivo de facilitar a coleta, armazenamento e distribuição de erva-flores. O projeto tem como objetivo oferecer uma plataforma de coleta e armazenamento de erva-flores para a comunidade, permitindo que as pessoas possam coletar e armazenar erva-flores de forma fácil e segura.
